# Amazon 广告投放策略

面向 Amazon 卖家的广告策略 skill。
帮助你快速生成一套可执行的投放方案，覆盖 Sponsored Products、Sponsored Brands、Sponsored Display，并把预算分配、广告结构、关键词策略、否词管理和 ACoS/TACoS 优化节奏一次讲清楚。

**适用平台：** Amazon、Shopify、WooCommerce、Walmart、TikTok Shop、Etsy、eBay、BigCommerce。

**适合人群：** Amazon 运营、新手投手、独立站卖家、电商顾问、品牌团队。

**核心价值：**
- 不只给方向，直接给执行建议
- 不只讲广告类型，还给预算和结构拆法
- 不只说优化，还告诉你先做什么、后做什么

## 使用指南

### 1. 这个 skill 是做什么的

这个 skill 用来帮你快速生成一套可执行的 Amazon 广告投放方案。
如果你不知道广告该怎么分 Campaign、预算怎么切、关键词怎么挖、ACoS 怎么降，可以直接用它。

它特别适合以下场景：
- 刚开始做 Amazon 广告，不知道从哪里下手
- 已经在投广告，但 ACoS 太高
- 有新品要启动，不知道前期预算和结构怎么配
- 想把零散的投放动作整理成一套完整策略

### 2. 你需要怎么使用

你只需要直接告诉模型你的业务信息，越具体越好。
至少建议带上以下信息：
- 产品是什么
- 卖哪个站点，例如 US / UK / DE
- 有几个 SKU
- 月预算或日预算
- 当前广告表现，例如 ACoS / TACoS / CPC / CTR / CVR
- 你的目标，例如降 ACoS、提销量、推新品、做品牌词防守

如果信息不完整，skill 会只追问一次，把缺的信息一次性问完。

### 3. 推荐提问模板

不会写需求时，可以直接复制下面这段：

```text
请为我的 Amazon 产品制定广告投放策略。
产品类目：
站点：
SKU 数量：
产品阶段：新品 / 成长期 / 稳定期 / 清库存
当前月预算：
当前核心指标：
当前广告问题：
目标：
```

### 4. 你会得到什么结果

正常情况下，skill 会输出 8 个部分：
- 一、结论摘要：先告诉你应该怎么投
- 二、现状判断：判断你现在的问题在哪里
- 三、广告结构建议：告诉你 Campaign 应该怎么拆
- 四、预算分配建议：告诉你不同广告类型怎么分预算
- 五、关键词与否词策略：告诉你怎么挖词、收词、否词
- 六、ACoS / TACoS 与优化节奏：告诉你看什么指标、多久调一次
- 七、优先级行动清单：告诉你先做什么、后做什么
- 八、下一步：告诉你接下来应该继续补哪些动作

### 5. 新手最常见的使用方法

#### 场景 A：完全不会投广告
直接提问：

```text
我刚开始做 Amazon 广告，请帮我给这款产品设计一套从 0 到 1 的广告方案。产品是便携榨汁杯，美国站，月预算 1500 美元，目标是先稳定出单。
```

#### 场景 B：已经在投，但 ACoS 太高
直接提问：

```text
我现在 Amazon 广告 ACoS 35%，目标降到 25%。产品是厨房收纳类，共 5 个 SKU，月预算 3000 美元。请帮我重做广告结构和优化节奏。
```

#### 场景 C：新品冷启动
直接提问：

```text
请给我一套 Amazon 新品广告启动方案。产品是宠物除毛刷，美国站，新品刚上架，预算每天 50 美元，目标是前 30 天完成测词和拿到第一批稳定订单。
```

### 6. 使用时要注意什么

- 不要只说“帮我优化广告”，信息太少会导致建议偏泛
- 最好提供真实预算和当前指标，结果会更准
- 如果你没有 TACoS、CVR 等数据，也可以先只给 ACoS、预算、产品信息
- 如果 skill 输出里出现 `⚠️`，说明这一部分是基于缺失信息做的估算
- 这个 skill 给的是策略和执行建议，不会自动替你修改 Amazon 后台广告设置

### 7. 最推荐的使用顺序

对于新手，建议按这个顺序使用：
- 第一次：先让 skill 帮你搭广告结构
- 第二次：拿现有数据让 skill 帮你看关键词和否词
- 第三次：再根据最近 7 天或 14 天数据，让 skill 帮你调预算和出价

这样用，会比一次性问一个很大的泛问题更容易落地。

## 常见问题

### 1. 我没有很多数据，也能用吗

可以。哪怕你只有产品、站点、预算和目标，也可以先用这个 skill。
如果数据不足，skill 会在必要时补问一次，或者用 `⚠️` 标记估算结论。

### 2. 我不是专业广告投手，也能看懂结果吗

可以。这个 skill 的输出已经按业务执行顺序拆开，你只需要优先看：
- 广告结构建议
- 预算分配建议
- 优先级行动清单

### 3. 它会自动帮我改 Amazon 广告后台吗

不会。它负责给你策略、结构和执行建议，实际广告创建和调整需要你自己到后台操作。

### 4. 一次提问越长越好吗

不是。只要把关键业务信息说清楚就够了。
重点是产品、站点、预算、当前表现和目标，不需要堆很多无关描述。

## 维护说明

### 1. 这个 skill 的目标

这个 skill 不是泛泛输出广告建议，而是要把用户输入整理成一套可执行的 Amazon 广告策略。
核心目标是：少空话、少理论、直接给出结构、预算、关键词、否词和优化节奏。

### 2. 输入设计原则

开发或维护这个 skill 时，输入侧应优先识别这些字段：
- 产品或产品线
- 类目
- 平台 / 站点
- 产品阶段
- 当前投放情况
- 当前核心指标
- 预算
- 业务目标

如果信息不完整，只允许补问一次，避免多轮来回消耗。

### 3. 输出设计原则

输出必须满足以下要求：
- 结构固定，方便用户阅读和复用
- 建议可执行，不能只给方向
- 能给 benchmark 时尽量给区间
- 信息不足时明确标注 `⚠️`
- 优先面向业务执行，不写成理论教材

### 4. 推荐的提示链路

建议保持以下链路不变：
- 先提取用户已知信息
- 再一次性补齐缺失信息
- 再按固定模块完成分析
- 最后按固定 8 段结构输出

### 5. 这个 skill 为什么这样设计

这样设计的目的有三个：
- 降低新手使用门槛
- 提高输出稳定性
- 让结果更容易直接转成实际投放动作

### 6. 维护时不要轻易改动的部分

以下部分建议保持稳定：
- “只补问一次”的规则
- 固定的分析模块
- 固定的输出结构
- `⚠️` 估算标记规则

这些约束会直接影响输出质量和一致性。

## 能力范围

- 设计全漏斗广告策略（SP + SB + SD）
- 设计 Campaign 结构（auto / broad / phrase / exact）
- 分配不同广告类型和层级的预算
- 按产品生命周期设定 ACoS / TACoS 目标
- 制定关键词挖掘与否词管理流程
- 提供出价优化方法
- 提供新品广告启动打法

## 工作方式

**步骤 1：提取已知信息**

先从用户输入中识别并整理以下信息：
- 产品或产品线
- 所属类目
- 销售平台
- 产品阶段（新品 / 成长期 / 稳定期 / 清库存）
- 当前广告投放情况
- 当前核心指标（如 ACoS、TACoS、CTR、CVR、CPC、ROAS）
- 月预算或日预算
- 业务目标（如控 ACoS、提销量、推新品、打品牌、做再营销）

**步骤 2：如信息不足，只补问一次**

如果缺少关键分析信息，只允许发起一轮补充提问。
要求：
- 将所有缺失问题合并到同一条消息中
- 优先使用多选题或结构化问题
- 支持用户使用简写回答，例如：`1b 2c 3a`
- 不要连续追问多轮

**步骤 3：完成策略分析**

分析时必须覆盖以下模块：
1. 广告组合策略
   - SP、SB、SD 各自承担什么目标
   - 不同广告类型的协同关系
2. Campaign 结构设计
   - auto / broad / phrase / exact 的分层方式
   - 品牌词、类目词、竞品词的拆分建议
3. 预算分配
   - 各广告类型预算占比
   - 测试预算与放量预算分配
4. 指标目标
   - 根据产品阶段给出合理的 ACoS / TACoS 目标
5. 关键词策略
   - 关键词来源
   - 扩词逻辑
   - 收词逻辑
6. 否词管理
   - 否词标准
   - 否词节奏
7. 出价优化
   - 调价时机
   - 调价依据
   - 高转化词和低效词的处理方法
8. 新品打法（如适用）
   - 冷启动阶段广告策略
   - 数据积累与放量路径

**步骤 4：输出最终结果**

输出必须具体、结构化、可执行，不要给空泛建议。

## 输出格式

请使用以下输出结构：
- 一、结论摘要
- 二、现状判断
- 三、广告结构建议
- 四、预算分配建议
- 五、关键词与否词策略
- 六、ACoS / TACoS 与优化节奏
- 七、优先级行动清单
- 八、下一步

补充规则：
- 尽量给出 benchmark、经验区间或合理范围
- 如果关键信息不足，明确使用 `⚠️` 标记该判断为估算
- 不要只说“建议优化”或“建议测试”，必须说明优化什么、怎么测、先后顺序是什么
- 输出面向业务执行，不要写成理论课或教材
